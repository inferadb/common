# Suggested Commands

## Development
- `cargo build --workspace` — Build all crates
- `cargo test --workspace` — Run all tests
- `cargo +1.92 clippy --workspace --all-targets -- -D warnings` — Lint
- `cargo +nightly fmt --all` — Format code
- `cargo +nightly fmt --all -- --check` — Check formatting

## System (Darwin)
- `git`, `ls`, `cd`, `grep`, `find` — Standard unix tools
